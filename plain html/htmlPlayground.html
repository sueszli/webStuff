  • strikethrough
  • underline
  • mark

Interactive stuff

The <form> element is a container for different types of <input> elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.
Forms can be used to perform HTTP requests.

Forms

If used without JavaScript then the default behaviour is to send a post request to a page called /action_page.php with the query .../action_page.php?fname=John&lname=Doe.
